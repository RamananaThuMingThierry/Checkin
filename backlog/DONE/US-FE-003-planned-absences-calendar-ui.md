# US-FE-003 - Exposer le calendrier RH des absences planifiees

## User Story

En tant que RH ou manager, je peux consulter un calendrier React des absences planifiees afin d'anticiper les indisponibilites.

## Valeur

Transformer les conges approuves et jours feries en vue planning lisible.

## Dependances

- depend de `US-040`
- depend de `US-FE-001`

## Scope frontend

- ecran calendrier RH
- filtres de periode et perimetre
- affichage des evenements `approved_leave` et `holiday`

## Criteres d'acceptation

- les absences planifiees sont visibles sur une plage donnee
- les jours feries apparaissent dans la meme vue
- les filtres metier sont appliques

## Verification

- build frontend reussi
- verification manuelle du calendrier
